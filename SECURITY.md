# Security

This repository is a sanitized public extract. It should not contain production credentials, private client data, internal hostnames, private IP addresses, or operational secrets.

If you find sensitive data in this repository, please report it privately to the repository owner.

## Public Demo Rules

- Use placeholder configuration only.
- Keep secrets in environment variables or local secret managers.
- Do not commit .env files, tokens, certificates, customer exports or production logs.
- Treat all external API credentials as private.
